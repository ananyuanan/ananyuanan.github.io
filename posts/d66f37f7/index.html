Github开源软件分享

1.阅读

阅读是一款可以自定义来源阅读网络内容的工具，为广大网络文学爱好者提供一种方便、快捷舒适的试读体验。

  • 自定义书源，自己设置规则，抓取网页数据，规则简单易懂，软件内有规则说明。
  • 书源规则支持搜索及发现，所有找书看书功能全部自定义，找书更方便。
  • 订阅内容,可以订阅想看的任何内容,看你想看。
  • 支持替换净化，去除广告替换内容很方便。
  • 支持本地TXT、EPUB阅读，手动浏览，智能扫描。
  • 支持高度自定义阅读界面，切换字体、颜色、背景、行距、段距、加粗、简繁转换等。
  • 软件开源，持续优化，无广告。
  • 点击访问阅读仓库
  • 点击访问阅读APP源 - AOAOSTAR

2.PicaComic

免费观漫软件，拥有大量的全彩和黑白漫画资源，emmmm，比如说18X。

  • 可以将喜欢的漫画作品进行离线缓存，方便无网络连接时进行阅读。

  • 可以提供漫画更新的提醒功能，及时了解到喜欢的作品的最新章节。

  • 软件提供无广告的阅读环境，专心阅读漫画，不会被广告弹窗打扰。

  • 点击访问PicaComic仓库

3.Simple live

简简单单的看直播，好用！

  • 支持Android、 iOS、 Windows 、 MacOS 、Linux 、Android TV 。
  • 支持虎牙直播、斗鱼直播、哔哩哔哩直播以及抖音直播。
  • 点击访问simple_live仓库

4.影视

影视APP是基于TVBOX二改来的一款影视盒子,分为两个版本，一个是ok版，一个是fongmi版，两个版本数据互通，目前我个人使用的是OK版。

关于接口，打开饭总网站，提供多个接口可供选择。

注：再次声明，影视不是tvbox！影视代码是由蜂蜜从无到有的！绝对不是从tvbox代码沿用！

来自-2024.7.13 影视TV TG频道

  • 支持Android、Android TV 。
  • 播放流畅度和清晰度可调。
  • 简约洁净，没有广告，现在版本增加了AI去广告。
  • 源更新比较快，资源也比较全。
  • 可看IPTV。
  • 点击访问影视ok版仓库
  • 点击访问影视fongmi版仓库
  • 点击访问饭太硬接口

5.gkd

搞快点搞快点搞快点！

基于无障碍，高级选择器，订阅规则的自定义屏幕点击 Android 应用，用于跳软件开屏广告等。

  • 随时随地自定义设置跳广告规则。
  • APP随时使用无需连接互联网，也不用手机ROOT，从源头杜绝隐私泄露的可能。
  • 无需复杂的设置就可以轻松跳过广告。
  • 点击访问gkd仓库

6.UnblockNeteaseMusic

一句话介绍：解锁网易云音乐客户端变灰歌曲~

  • 支援多个音源，替换变灰歌曲链接.

  • 为请求增加 X-Real-IP 参数解锁海外限制，支持指定网易云服务器 IP，支持设置上游 HTTP / HTTPS 代理。

  • 完整的流量代理功能 (HTTP / HTTPS)，可直接作为系统代理 (同时支持 PAC)。

    注：可能会存在失效问题，如失效，找fork修复版。

（1）Windows

  • 推荐直接下载releases版本，每次使用网易云之前先打开unblockneteasemusic，再打开网易云音乐。

  • 有余力的可以安装node，然后参考网站的做法，我个人采用的是这种方法，每次使用只需要打开网易云就可以了。

  • 注：记得先安装证书ca.crt。

  • 点击访问UnblockNeteaseMusic

（2）安卓版本-杜比大喇叭β版

  • 需要root，安装lsposed框架，然后安装模块勾选。
  • 不需要root，直接下载内嵌版。
  • 请点击Actions，下载最新版本。
  • 点击访问杜比大喇叭

（3）更省事版本-YesPlayMusic

  • 高颜值的第三方网易云播放器。

  • 支持 UnblockNeteaseMusic，自动使用各类音源替换变灰歌曲链接 （网页版不支持）。

  • 支持 Windows / macOS / Linux 。

  • 点击访问YesPlayMusic
